# Mini Banking System

A Mini Banking System application developed using Java for backend logic and HTML, CSS, and JavaScript for the frontend UI.

## Features
- Create a bank account
- Deposit money
- Withdraw money with validation
- Check current account balance
- Simple and user-friendly frontend interface

## Technologies Used
- Java (OOP, validation, exception handling)
- HTML
- CSS
- JavaScript

## Project Structure
- `MiniBankingSystem/` – Java backend logic
- `Bank-ui/` – Frontend UI files

## How to Run the Project

### Backend (Java)
1. Open the `MiniBankingSystem` folder
2. Compile the program: javac Main.java
3. Run the program: java Main


### Frontend
1. Open the `Bank-ui` folder
2. Double-click `index.html` to run in a browser
